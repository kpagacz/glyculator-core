Fires up cleaning routines

Performs cleaning operations on the CGM file. Removes rows with missing dates, empty rows, duplicated rows, rows with glucose value between the regularly interspersed measurements. Uses information stored in the CleanConfig object passed to the FileCleaner on init.

Returns

date time column and glucose values column. Returns a cleaned CGM.

Return type

A dataframe with two columns

Raises

RuntimeError – if untidy attribute is None if clean_config attribute is None
